# 个人猎头网站 - 需求规格文档

> 猎头个人品牌网站，展示合作团队和岗位合集

---

## 1. 项目概述

- **项目名称**: HeadHunter Portfolio
- **项目类型**: 个人品牌展示网站
- **核心功能**: 展示猎头个人介绍 + 行业/公司/岗位三级目录 + 岗位详情页
- **目标用户**: 候选人、企业 HR、合作伙伴

---

## 2. UI/UX 设计

### 2.1 视觉风格

**Illustration-rich UI** - 手绘插图风格
- 温暖米白背景 `#F7F6F3`
- Notion/Linear 风格：友好、简洁、有温度
- 手绘插图作为空状态和英雄区装饰
- 涂鸦风格的分隔符
- 插图图标

### 2.2 色彩系统

```css
--bg-primary: #F7F6F3;      /* 温暖米白 */
--bg-card: #FFFFFF;         /* 卡片白 */
--text-primary: #1A1A1A;    /* 主文字 */
--text-secondary: #6B7280;  /* 次要文字 */
--accent: #FF6B35;          /* 活力橙 - 强调色 */
--accent-light: #FFF0EB;    /* 浅橙背景 */
--border: #E5E7EB;          /* 边框 */
```

### 2.3 字体

- **标题**: `Clash Display` 或 `Cabinet Grotesk`（有特色的现代字体）
- **正文**: `Satoshi` 或 `General Sans`
- **中文**: `Noto Sans SC`

### 2.4 布局

- **首页**: 英雄区(个人介绍) → 行业分类 → 公司列表
- **内页**: 公司详情 → 岗位列表 → 岗位详情
- **响应式**: 移动端单栏，桌面端多栏

### 2.5 组件

| 组件 | 描述 |
|------|------|
| HeroSection | 个人头像、姓名、 tagline、联系方式 |
| IndustryCard | 行业卡片（如具身智能、AI） |
| CompanyCard | 公司卡片（名称、logo、简介） |
| JobCard | 岗位卡片（标题、标签、薪资） |
| ProfileBadge | 画像标签（薪资/资历/语言） |
| EmptyIllustration | 空状态手绘插图 |
| CursorDecorations | 鼠标跟随涂鸦装饰 |

## 5. 交互动效

- **页面切换**: 平滑淡入 + 轻微上移
- **卡片悬停**: 轻微上浮 + 阴影加深
- **按钮点击**: 轻微缩放反馈
- **加载状态**: 骨架屏 + 手绘加载插图
- **空状态**: 友好的手绘插图 + 引导文字

---

## 6. 技术栈

- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS + 自定义 CSS 变量
- **数据**: 本地 JSON 文件
- **动画**: Framer Motion
- **图标**: Lucide React
- **字体**: Google Fonts (Noto Sans SC) + 本地字体
